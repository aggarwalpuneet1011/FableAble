# FableAble 📖✨

A branching, AI-generated bedtime and travel story app for kids. Built with React Native (Expo SDK 54) in a single `App.js` file.

The child picks a hero name, what they love to do, and a genre — then chooses from 3 options at each of 5 story chapters. Every story is freshly generated by Gemini.

---
## Why

- Built for my daughter, 7yrs old. I'd built her typing tutors and story tools that she loves — all running on a laptop she can't access stuck in a traffic jam in Sakinaka. This is the same thing, minus the laptop dependency.
---
## Features

- 🤖 **AI-generated stories** via Google Gemini — every adventure is unique
- 🎭 **8 genres** — Fantasy, Superhero, Space, Ocean, Animals, Magic School, Dinosaurs, Princess
- 🔀 **Branching choices** — 3 options per chapter across 5 parts
- ← **Go back** — change your mind and take a different branch
- 🔊 **Read aloud** — text-to-speech on every chapter (great for pre-readers)
- ✈️ **Offline mode** — 2 bundled stories play with no internet
- 💾 **Auto-resume** — picks up where you left off if the app closes
- 💬 **Share** — export the full story as a text file

## Setup

### Requirements

- Node.js v20.x (use [nvm-windows](https://github.com/coreybutler/nvm-windows) on Windows)
- [Expo Go](https://expo.dev/go) app on your phone (for development)
- A free [Google Gemini API key](https://aistudio.google.com/app/apikey)

### Install and run

```bash
# 1. Clone the repo
git clone https://github.com/aggarwalpuneet1011/FableAble.git
cd FableAble

# 2. Install dependencies
npx expo install expo-secure-store expo-file-system expo-sharing expo-network expo-speech expo-haptics @react-native-async-storage/async-storage react-native-safe-area-context expo-status-bar

# 3. Start
npx expo start
# Scan QR with Expo Go on your phone
```

### API Key

The app asks for your Gemini API key on first launch (tap the ⚙️ gear icon). The key is stored securely on-device using Android Keystore — it is never in the code or sent anywhere except directly to Google's API.

Get a free key at [aistudio.google.com](https://aistudio.google.com/app/apikey).

---

## Building an APK

Uses [EAS Build](https://docs.expo.dev/build/introduction/) (free tier, ~30 min/month):

```bash
npm install -g eas-cli
eas login
EAS_NO_VCS=1 eas build -p android --profile preview
```

Download the `.apk` from your [Expo dashboard](https://expo.dev) and install directly on Android.

## Project structure

Everything is in a single `App.js` file — all screens, logic, offline stories, and styles. `app.json` handles Expo configuration.

## Notes

- Default Gemini model: `gemini-3.5-flash` (configurable in-app via ⚙️ if Google retires it)
- Story cost: fractions of a cent per full 5-part story
- Content safety: strict Gemini safety settings + prompt guardrails keep stories age-appropriate
